# Setting up your laptop

## Hardware

IT assigns each teammate a laptop from the asset register. If something is
broken or missing, file a ticket — don't buy replacements yourself.

## Accounts

| Service        | How to get access               |
| -------------- | ------------------------------- |
| Email          | Created before your first day   |
| Password vault | Invitation from IT on day 1     |
| Git forge      | Ask your team lead              |
| Internal apps  | Sign in with your company email |

## Security basics

- Full-disk encryption stays ON (it ships enabled — don't disable it).
- The password vault generates and stores every password; nothing in plain
  text files.
- Updates install weekly — restart when prompted, not "later this month".

See [Security policy](/kb/security-policy) for the full rules.
